# 커리어 시뮬레이션 및 스탯 산출 로직 요약

## 1. 기본 스탯 (Basic Stats)
훈련생 시절 육성하는 6가지 기초 능력치입니다. (20~80 스케일)
- **근력 (Power)**
- **유연성 (Flexibility)**
- **스피드 (Speed)**
- **쓰로잉 (Throwing)**
- **배트 스피드 (Bat Speed)**
- **악력 (Grip)**

## 2. 야구 스탯 변환 (Baseball Skills)
포지션 배정(매칭) 시 기초 능력치를 조합하여 야구 스킬로 변환합니다.

### 투수 (Pitcher)
- **구위 (Stuff)**: `(쓰로잉 + 근력) / 2`
- **제구 (Control)**: `(악력 + 유연성) / 2`
- **변화 (Breaking)**: `(악력 + 배트 스피드) / 2`
- **구속 (Top Speed)**:
  - 기본 88마일에서 시작하여 구위 스탯에 따라 증가
  - 공식: `88 + (구위 - 20) * (11/60)`
  - 최대 107마일 (약 172km/h)까지 가능

### 타자 (Fielder)
- **파워 (Power)**: `(근력 + 배트 스피드 + 악력*0.5) / 2.5`
- **컨택 (Contact)**: `(배트 스피드 + 유연성 + 악력*0.5) / 2.5`
- **수비/주루 (Defense)**: `(스피드 + 쓰로잉) / 2`

## 3. 프로 커리어 시뮬레이션 (Pro Career Simulation)
'프로행(Send to Pro)' 선택 시 진행되는 시뮬레이션 로직입니다.

### 기본 설정
- **활동 기간**: 10년 ~ 20년 (랜덤)
- **시작 나이**: 20세

### 에이징 커브 (Aging Curve)
선수의 나이에 따라 능력치가 변화합니다.
1.  **성장기 (~26세)**: 현재 능력치에서 잠재력(Potential)을 향해 매년 성장
2.  **전성기 (27세 ~ 31세)**: 잠재력 만개 (최고 기량 유지)
3.  **쇠퇴기 (32세 ~)**: 매년 능력치 하락 (연간 -2)

### 시즌 성적 산출 (Season Stats)
해당 시즌의 능력치(Rating)를 기반으로 성적을 생성합니다. (약간의 랜덤 변수 포함)

#### 투수
- **평균자책점 (ERA)**: `7.5 - (Rating / 13)` (낮을수록 좋음, 최저 1.50)
- **승리 (Wins)**: `Rating * 0.25`
- **탈삼진 (SO)**: `Rating * 2.2`

#### 타자
- **타율 (AVG)**: `0.180 + (Rating / 350)` (최대 0.400 근접)
- **홈런 (HR)**: `Rating * 0.6`
- **OPS**: 타율 + 장타율 공식을 약식 적용

### 수상 기준 (Awards)
매 시즌 성적에 따라 수여됩니다.
- **MVP**:
  - 투수: 18승 이상 & ERA 2.50 미만
  - 타자: 40홈런 이상 또는 타율 0.340 이상
- **사이영 (Cy Young)**: 15승 이상 & ERA 3.00 미만
- **골든글러브 (Golden Glove)**: 30홈런 이상 또는 타율 0.320 이상
- **신인왕 (Rookie of the Year)**: 1년차에 10승 이상(투수) 또는 20홈런/타율 0.280 이상(타자)

## 4. 보상 (Rewards)
명예의 전당 등록 시 획득하는 XP입니다.
- **기본 보상**: 1000 XP
- **추가 보상**: 수상 횟수 당 +1000 XP
